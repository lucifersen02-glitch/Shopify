# Shopify Product Setup Guide - Hexagon Mirror Stickers

## Step-by-Step Instructions

### 1. Product Description
Copy the description from `product-description-template.md` and paste it into the Rich Text Editor field.

### 2. Category Selection
- Go to "Category" dropdown
- Select: **Home Decor** or **Wall Decor**
- This helps with:
  - Proper tax calculation
  - Better search results
  - Organized product display

### 3. Media/Images
✅ Already uploaded - Make sure:
- Main product image is set as featured image
- All variant images are uploaded
- Images show the product clearly

### 4. Variants Setup
Your variants look good:
- **Size:** 6 pieces / 12 pieces
- **Color:** (if applicable)
- Prices are set correctly
- Inventory is set (50,000 each)

### 5. Search Engine Listing (SEO)
Click "Edit website SEO" and fill in:

**Page title:**
```
Hexagon Acrylic Mirror Wall Stickers 6/12Pcs | Home Decor DIY | PerfectFinds
```

**Meta description:**
```
Shop hexagon acrylic mirror wall stickers - 6 or 12 pieces. Removable DIY home decor stickers perfect for living room decoration. Easy installation, modern design. Free shipping!
```

**URL:**
```
hexagon-acrylic-mirror-wall-stickers-6-12pcs
```
(Shopify auto-generates this, but you can customize it)

### 6. Product Organization

**Type:** Home Decor

**Vendor:** Perfect Finds

**Collections:** 
- Add to "Home Decor" collection
- Create "Wall Decor" collection if it doesn't exist
- Add to "Home page" collection (already added)

**Tags:** Add these tags (comma-separated):
```
hexagon mirror stickers, wall decals, home decor, geometric pattern, removable stickers, DIY decor, mirror wall art, living room decor, bedroom decoration, modern home decor, acrylic mirrors, wall stickers, home improvement, interior design, accent wall, removable decor
```

### 7. Sales Channels
- ✅ Enable "Online Store" 
- Make sure product is visible in your store

### 8. Status & Publishing

**Before Publishing:**
1. Review all information
2. Check images display correctly
3. Verify prices
4. Test variant selection
5. Check description formatting

**To Publish:**
1. Click "Save" first
2. Then click "Publish" button (top right)
3. Or change status from "Draft" to "Active"

### 9. Additional Recommendations

**Product Images:**
- Main image: Product on white background
- Image 2: Product installed on wall
- Image 3: Close-up detail
- Image 4: Multiple pieces arranged
- Image 5: Room setting/styled photo

**Product Options:**
Consider adding:
- Installation guide PDF
- Size chart/dimensions image
- Care instructions

**Marketing:**
- Create a discount code for first-time buyers
- Add to featured products section
- Create a blog post about wall decor trends
